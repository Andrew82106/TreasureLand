# 《万物之岛》Godot 原型

这是依据 `../doc/` 当前设计文档制作的第一版纵向切片。它使用 Godot 内置节点与代码绘图，不依赖任何正式美术素材。

## 如何启动

1. 打开 Godot 4.x。
2. 点击“导入”，选择本目录下的 `project.godot`。
3. 打开项目后按 `F6` 或右上角运行按钮；主场景已设置为 `main.tscn`。

建议使用 Godot 4.4 或更高版本。本项目已用 Godot 4.7 stable 完成解析、无窗口启动和烟雾测试。

## 操作

- `WASD` 或方向键：移动。
- `E`：与附近地点或 NPC 互动。
- `I` 或 `Tab`：打开背包。
- `Esc`：关闭当前界面。
- 顶部“地图”：查看已发现区域并快速移动；移动与快速移动不消耗潮刻。

## 当前可玩的验证循环

1. 在漂流湾浅滩采集材料和金贝。
2. 把水、土等材料按顺序投入造化盆；材料正确时直接生成物品。
3. 在椰影杂货铺买卖，或在背包中锁定物品。
4. 合成鱼与盐得到咸鱼，交给逐风海岸的阿葵完成委托。
5. 在椰影街参加水火占卜牌会，使用两张命牌与逐步公开的天象牌形成命象；牌桌右上角可以随时查询完整规则。
6. 在逐风海岸参加八兽四段竞速。
7. 回漂流小屋休息，刷新天气、商店和赛事。

原型中的万象塔是长期目标占位，目前不开放。NPC、商店、背包、地图、合成、牌会、竞速、金贝与潮刻共用同一份运行状态。

## 自动检查

仓库根目录的 `tools/ci/run_godot_tests.py` 是本地与 GitHub Actions 共用的阻断测试入口。Godot 在命令行可用时，从仓库根目录运行：

```powershell
python tools/ci/run_godot_tests.py --godot godot --project main
```

该入口先检查 Godot 4.7 stable、项目导入与脚本解析，再依次运行 CI 宪法登记的六个阻断测试。视觉截图脚本目前用于人工评审，不属于主门禁。
