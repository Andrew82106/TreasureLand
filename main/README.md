# 《万物之岛》Godot 原型

这是依据 `../doc/` 当前设计文档制作的纵向切片。场景、NPC、牌会、竞速与合成已接入首轮正式或占位美术，规则与界面仍在持续细化。

## 如何启动

1. 打开 Godot 4.x。
2. 点击“导入”，选择本目录下的 `project.godot`。
3. 打开项目后按 `F6` 或右上角运行按钮；主场景已设置为 `main.tscn`。

建议使用 Godot 4.4 或更高版本。本项目已用 Godot 4.7 stable 完成解析、无窗口启动和烟雾测试。

## 操作

- `WASD` 或方向键：移动。
- `E`：与附近地点或 NPC 互动。
- `I` 或 `Tab`：打开万物图鉴。
- `Esc`：关闭当前界面。
- 顶部“地图”：查看已发现区域并快速移动；移动与快速移动不消耗潮刻。

## 当前可玩的验证循环

1. 在漂流湾浅滩观察潮汐并获得有限的金贝收入。
2. 进入独立造化盆页面，从水、火、土开始进行永久二元关系实验，沿100%、70%、40%的层间关系覆盖逐步发现72种万物与69条四阶关系。
3. 在椰影杂货铺购买组合方向线索或实验折扣；商店不会直接出售万物。
4. 合成陶器与水得到水罐，向逐风海岸的阿葵分享补水方法。
5. 在椰影街参加水火占卜牌会，使用两张命牌与逐步公开的天象牌形成命象；牌桌右上角可以随时查询完整规则。
6. 在逐风海岸参加八兽四段竞速。
7. 回漂流小屋休息，刷新天气、商店和赛事。

万象塔按四阶关系进度逐层点亮。NPC、商店、万物图鉴、地图、合成、牌会、竞速、金贝与潮刻共用同一份运行状态。

## 自动检查

仓库根目录的 `tools/ci/run_godot_tests.py` 是本地与 GitHub Actions 共用的阻断测试入口。Godot 在命令行可用时，从仓库根目录运行：

```powershell
python tools/ci/run_godot_tests.py --godot godot --project main
```

该入口先检查 Godot 4.7 stable、项目导入与脚本解析，再依次运行 CI 宪法登记的七个阻断测试。视觉截图脚本目前用于人工评审，不属于主门禁。
