# 《万物之岛》金贝与基础经济设计 v0.1

> 文档状态：原型实施基线
> 货币名称：金贝
> 核心目标：让探索、信息与风险活动共享同一套可解释的财富循环

## 1. 经济定位

金贝是唯一通用货币。玩家使用金贝进行万物实验、购买研究服务、部署竞速辅助、参加牌会和购买祝胜券。

永久万物属于知识、能力与收藏，不计数量、不进入买卖价格，也不计入净资产。金贝经济只记录真实账户资金与活动中被锁定的本金。

## 2. 账户口径

### 2.1 当前金贝

玩家可以立即使用的金贝。

### 2.2 活动中金贝

已经投入牌会或竞速、尚未完成结算的本金。活动中金贝不能同时用于其他消费。

### 2.3 账户财富

```text
账户财富 = 当前金贝 + 活动中金贝 + 份契待撮合买单冻结金贝
```

### 2.4 净资产

接入海岸潜捕后，鱼获属于可以出售的短期商品资产；永久万物仍不计价：

```text
净资产 = 当前金贝 + 活动中金贝 + 待撮合金贝 + 鱼获保守变现值 + 份契保守变现值
```

鱼获保守变现值使用当前鱼铺批量出售预览的实际可得金额，不使用单条最高报价简单相乘。财富头衔与公共资格仍读取账户财富，不读取鱼获估值，防止短期价格波动反复开关资格。

份契保守变现值使用当前报价的98%，反映卖出交易费和价格风险。份契不伪装成当前可用金贝；商会交易资格在账户财富首次达到20,000金贝后永久保留，但牌桌、赛事和消费仍检查真实可用现金。

永久图鉴、海生图鉴记录、关系、资格、免费体验券和未兑现信息不折算成金贝。

## 3. 财富头衔

| 头衔 | 账户财富门槛 | 主要开放内容 |
|---|---:|---|
| 初来者 | 0 | 基础实验、低额竞速、教学牌会 |
| 学徒 | 500 | 入席券、椰影常桌、更多研究服务 |
| 小赢家 | 3,000 | 风灯高桌、进阶知识与收藏陈列 |
| 富商 | 20,000 | 塔影名流桌、长期资产入口 |
| 大亨 | 150,000 | 高级服务与岛上投资预留 |
| 岛之名流 | 1,000,000 | 终局财富身份与长期影响力预留 |

头衔按当前账户财富即时升降。已经永久发现的万物和已经购买的研究服务不会因头衔下降而被回滚。

## 4. 金贝来源

### 4.1 稳定来源

| 来源 | 首版规模 | 限制 |
|---|---:|---|
| 海岸潜捕 | 普通表现每次 40—75 | 每日 3 个有效鱼群窗口，每次消耗 1 潮刻 |
| NPC 委托 | 40—150 | 每个委托首次完成一次 |
| 剧情与教学 | 固定 | 明确标注，不可重复领取 |
| 商会分红 | 取决于经营利润与持有期 | 只向满足持有期的份契发放；无利润时可为0 |

### 4.2 风险来源

| 来源 | 特点 | 约束 |
|---|---|---|
| 命运牌会 | 玩家与 NPC 之间转移资金 | 服务费、桌级上限、独立教学额度 |
| 逐风竞速 | 依据信息和赔率进行风险判断 | 单场投入上限、组织方返还率、公开结算 |

### 4.3 不产生金贝的成长

- 发现新万物。
- 图鉴与万象塔进度。
- NPC 关系与记忆。
- 商店购买的线索与研究优惠。
- 竞速辅助带来的额外信息。

这些成长提供选择权和长期目标，不直接伪装成现金资产。

## 5. 金贝消耗

### 5.1 万物实验

| 最高输入阶层 | 标准费用 |
|---:|---:|
| 一阶 | 2 |
| 二阶 | 5 |
| 三阶 | 12 |
| 四阶 | 30 |

- 只对第一次尝试的组合收费。
- 已知成功或已知无效组合再次查看费用为 0。
- 金贝不足时不建立尝试记录。
- 限次折扣只在真实新尝试时消耗。
- 真正执行的新组合在原子结算完成后固定消耗0.25潮刻；免费复查不追加时间成本。

### 5.2 商店研究服务

- 配方方向线索：12 金贝。
- 三次实验半价服务：24 金贝。
- 后续可增加地图知识等研究服务；收藏陈列由居所系统直接读取永久图鉴，不通过商店出售。
- 商店不能直接解锁谱系万物；全部万物由造化盆关系发现。

### 5.3 逐风竞速

- 祝胜券投入：最低 10 金贝。
- 信息型造物部署费：4—8 金贝。
- 免费体验券只免除固定 10 金贝票面，不免除自选辅助部署费。

### 5.4 命运牌会

- 使用对应桌级的局内上限与基础投入。
- 首次教学使用独立 100 金贝额度，不损害玩家原有账户。
- 正式牌会使用`activity_snapshot`：短牌会离桌固定推进2潮刻，完整牌会固定推进4潮刻，不叠加牌局演出与思考时间。

### 5.5 商会份契

- 即时和次日撮合买卖均收取1%交易费，向上取整且至少1金贝。
- 当日买入次日方可卖出。
- 次日买单冻结的金贝仍属于账户财富，但不能重复用于消费。
- 价格、经营报告、分红与持仓进入版本7存档和财富轨迹。

## 6. 开局经济

### 6.1 起始状态

- 当前金贝：120。
- 永久万物：水、火、土。
- 免费逐风体验券：1。
- 首次低桌牌会教学额度：100。

### 6.2 第一日可承担范围

开局资金应支持：

- 完成全部三组根关系并继续进行多次二阶实验；或
- 购买多个关系线索或实验折扣；或
- 多次低额竞速体验；或
- 组合上述活动并保留恢复金。

海岸潜捕的每日收益应足以覆盖数次基础实验和一次低额风险活动，但不能形成无限点击收入。少见或稀有鱼允许产生高于普通区间的单次惊喜，不能稳定抬高长期保底收益。

### 6.3 建议恢复金

界面提供建议留存金贝，帮助玩家理解风险，不阻止消费：

```text
建议恢复金 = max(200, 账户财富 × 5%)
```

## 7. 财富轨迹

财富轨迹在以下结算后记录节点：

- 新组合实验付费。
- 商店知识或服务付费。
- 竞速辅助部署、下注和派彩完成。
- 牌会离桌结算。
- 潜捕获得鱼获、鱼获出售、订单交付、委托与剧情奖励。
- 份契买卖、隔夜撮合、撤单返还、每日估值变化与分红。

每个节点至少保存：

```yaml
day: 1
tide: 3
reason: 万物实验 · 水 + 土
cash: 118
locked_principal: 0
net_worth: 118
delta: -2
```

潜捕上岸时以鱼获保守变现值记录资产增加；出售时记录“鱼获减少、现金增加”的资产转换，只有价格变化、尺寸溢价与舍入造成的真实差额才形成净值变化。同一业务行为不得被拆成多个无法解释的曲线节点。竞速下注和返还应在比赛结算后形成一个净变化节点。

## 8. 亏损与恢复

- 不暗改比赛结果或牌局发牌帮助玩家回本。
- 不根据财富提高万物实验基础费用。
- 海岸潜捕始终保留有限、稳定、可理解的恢复来源。
- 金贝不足时，免费查看图鉴、历史组合、公开赛事情报和地图。
- 已永久解锁的万物、知识和竞速辅助不因贫穷而失去。
- 风险活动必须在确认前显示投入上限和当前金贝。

## 9. 经济守恒

### 9.1 必须守恒的活动

- 牌会：六席投入 = 主池 + 边池 + 返还 + 服务费。
- 商店：玩家扣款必须对应一次知识解锁或服务交付。
- 万物实验：每个首次组合最多扣款一次。
- 竞速：账户变化 = 票面投入 + 辅助部署费 + 派彩。
- 鱼市：移除的鱼获 = 鱼铺新增库存或订单消耗；玩家所得金贝 = 出售预览确认金额。
- 商会份契：买入扣款 = 成交额 + 费用；卖出所得 = 成交额 - 费用；隔夜冻结现金 = 待撮合买单冻结总额；持仓数量 = 买入与撮合增加 - 卖出减少。

### 9.2 明确金贝去向

万物实验费、商店服务费、竞速辅助部署费和赛事组织方优势属于世界经济回收，不进入其他 NPC 钱包。鱼铺收购款属于世界消费与产业采购支出，必须对应真实库存、订单或加工需求。日志必须明确标记其来源，不能无记录消失。

## 10. 调试指标

每个存档和测试版本至少可统计：

- 玩家每日金贝来源与消耗占比。
- 每次首次发现的平均实验成本。
- 无效组合率与重复查看率。
- 商店线索与折扣购买顺序。
- 竞速辅助使用率和使用后的下注表现。
- 各风险活动的实际返还率。
- 玩家跌破建议恢复金后的恢复用时。
- 各鱼种捕获量、出售量、库存、需求满足率、平均成交价和失鲜率。
- 单次潜捕平均鱼获价值、稀有鱼贡献占比与每日潜捕总收益。
- 金贝是否出现负数、重复扣款或无来源增加。

## 11. 自动测试契约

1. 永久万物的价值不计入净资产。
2. 新组合只扣一次实验费，左右交换不重复收费。
3. 商店研究服务只有在可交付时扣款，且不得直接解锁万物。
4. 限次折扣的扣款和次数严格对应。
5. 竞速辅助部署费单独记录，造物不被消耗。
6. 活动结算后不残留错误的活动中金贝。
7. 金贝不得为负。
8. 财富轨迹必须能够解释每次净变化。
9. 海岸潜捕达到每日窗口上限后不再产生鱼获、金贝或消耗时间。
10. 鱼获出售必须同时移除鱼获、增加金贝和增加鱼铺库存，且同一成交最多结算一次。
11. 批量出售使用实际收购深度估值，财富轨迹不得用最高单价虚增净资产。
12. 牌会与竞速的长期返还约束继续成立。
13. 份契同日价格固定，单日涨跌不超过12%，当日买入不能当日卖出。
14. 隔夜买单冻结与撤回严格守恒，订单最多撮合一次。
15. 份契估值进入净资产但不进入当前可用金贝；版本7存档保留持仓成本和历史。

## 12. 原型验收

- 永久万物不计入现金估值，图鉴进度以收藏和玩法价值衡量。
- 开局资金足以支持连续发现尝试。
- 实验、商店服务和竞速辅助形成三个清楚的非风险消耗口。
- 图鉴成长和金贝财富在界面上使用不同口径。
- 所有扣款在确认前可见，结算后进入财富轨迹。
- 自动测试能够发现重复扣款、重复奖励、资金负数和统计失控。
