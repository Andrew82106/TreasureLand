# 《万物之岛》原型试玩反馈与全面细化计划 v0.1

> 反馈来源：首轮连续自由试玩  
> 结论：核心财富循环成立，项目从“验证玩法存在”进入“纵向细化与内容生产”阶段。

## 1. 已确认的问题

1. 浅滩采集可以依赖按键速度无限刷鱼和金贝，挤压其他赚钱路线。
2. 竞速命中率低时只有输赢结论，没有过程反馈，未中奖体验空洞。
3. 命运牌会具备核心判断玩法，但画面、桌级成长和行动反馈仍不足。
4. 合成失败不知道原因，成功缺少发现演出、插图、收藏与长期目标，因此玩家没有主动试配方的动力。
5. 地图、万象塔、牌会与竞速仍带有明显灰盒感，不能支撑财富成长幻想。
6. 玩家有钱后缺少更高额度、更高身份和更多系统去承接财富。

## 2. 本轮立即落地

- 浅滩每日3次收益上限，次日刷新；额外点击不再获得资源或消耗时间。
- 合成使用左右双槽永久万物；首次组合支付试验费，成功与失败都进入历史并给出关系反馈。
- 牌会增加80、200、500、2,000金贝四档桌级，基础投入与NPC钱包同步缩放。
- 竞速增加正式主视觉和四段赛程回放。
- 地图接入连续海岛全景，万象塔改为核心视觉地标，并用图鉴数量点亮四层进度。
- 财富轨迹继续承担赚钱目标反馈，图鉴与万象塔承担合成目标反馈。
- 合成扩展为独立全屏玩法页，固定四阶、21个万物与18条可解释关系；左右图鉴独立滚动，中央舞台承载唯一结果反馈。
- 杂货铺以插图卡片提供组合方向线索和三次试验折扣，不直接出售万物。
- 竞速接入雨势推演、雷暴听兆、补水观察和河势比照四类造物情报；造物只提高信息质量，不改变既定赛果。
- 命运牌会接入月夜茶亭牌桌、5名NPC独立肖像、呼吸与思考聚焦、逐席行动停顿、人物对白气泡和天象翻牌动画。

## 3. 下一轮细化顺序

### P1：把现有核心玩法做深

- 命运牌会：继续补足金贝流动与赢家演出；在现有五名人物肖像、逐席思考和性格对白基础上，强化不同桌级的策略差异。
- 逐风竞速：赛兽近况、历史成绩、分段优势、观赛动画与赛事日程。
- 万物合成：以当前四阶谱系为基线扩充五阶关系、世界变化、组合历史、首次发现收藏、献塔与NPC需求。
- 商店与NPC：扩充研究服务、人物来源线索和财富变化后的对话；生活消费延后到资产系统成熟后。

### P2：让财富转化为长期资产

- 住宅、服装、家具、收藏室和逐风兽培养；
- 高额牌会、赛事席位与商会资格；
- 图鉴、塔层与人物关系共同解锁新服务，而不是只提高数字。

### P3：逐步引入商会份契（股票原型）

股票系统不直接复制现实证券软件，先以“岛上商会份契”进入：

1. 首版只提供3家可理解的产业：渔获、造物运输、逐风赛事；
2. 价格变化必须来自玩家能看到的天气、需求、赛事和NPC新闻；
3. 先实现日结买卖、持仓成本、公开新闻和财富曲线联动；
4. 禁止高频点击套利、无成本内幕答案和只涨不跌的教学资产；
5. 等现有经济来源与消耗稳定后再开放杠杆、分红和复杂订单。

## 4. 验收方向

- 玩家不能再把浅滩当作无限点击器；
- 每次合成失败或竞猜未中都能解释原因，并形成下一次尝试目标；
- 每种新造物都有发现演出、插图、图鉴关系与塔层进度；
- 财富跨过门槛后出现新桌级、新服务或可见生活变化；
- 正式素材提升氛围，但不遮挡按钮、数值和交互标记；
- 新系统优先连接现有时间、新闻、NPC和财富数据，不制造孤立小游戏。
